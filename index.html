AUTOMATED WATERING SYSTEM USING ARDUINO UNO | C++

San Pablo City, Laguna | May 2024
I built an automated plant watering system using soil moisture and NPK sensors to monitor soil conditions in real time. The system collects sensor data to determine when irrigation is needed, supporting healthier and more efficient plant care. Using an Arduino Uno, I programmed the irrigation logic in C++ to manage watering cycles and control water flow through relay modules. This setup enabled automated, responsive irrigation based on actual soil conditions rather than fixed schedules.
