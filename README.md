## ğŸ“ Steps to Create a GitHub Page

1. **Navigate to Settings**
   - Go to your repository on GitHub
   - Click on the **Settings** tab in the top navigation bar

2. **Access Pages Settings**
   - In the left sidebar, scroll down and click on **Pages**

3. **Configure the Source Branch**
   - Under "Build and deployment", find the **Source** section
   - Select **Main Branch** from the dropdown menu

4. **Save Your Changes**
   - Click **Save** to apply the settings
   - Wait a few minutes for GitHub to build and deploy your site

5. **Access Your Site**
   - Your site will be available with the generated link
